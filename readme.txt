Creating XML from a CSV file for a simple business rule:
There are five other objects in this folder besides this read.me file.
	Create XML From CSV.ps1 - the PowerShell Script.
	Simple Rule Source.csv - Sample data to convert to XML. 
	Simple Rule for Upload.xml - Sample output for XML.  
    People Analytics Business Rules Report.xml - for use with UKG People Analytics - Cognos BI.
	Create=ing XML From a CSV.pdf - full instructions from our blog post
	
You only need the Create XML From CSV and the simple rule Source file.
simply update the PowerShell Script with the name of your rule and run the program.  The XML will be created and you can proced with the upload.
full instructions can be found in the pdf.
